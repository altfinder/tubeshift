Thanks For Installing TubeShift!

Welcome to the browser extension that helps you watch less YouTube.

There are a few things that need to be setup and that you should know about before you get started.

Enable Permissions

TubeShift works better if you enable access to the video platform web pages. This is the same thing an ad blocker needs but gives you more control over which websites it will happen on.

How It Works

Look in the upper right hand side of your browser for the TubeShift arrow.

If you don't see the TubeShift arrow then click on the jigsaw puzzle piece icon and then pin TubeShift.

Click on the TubeShift arrow to see the user interface. This is where you can see the list of known alternative locations for the video you are watching as well as where you access the extension settings.
